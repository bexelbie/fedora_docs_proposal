Introduction to Boxes

Introduction to Boxes

Boxes is a simple graphical interface for managing and using virtual machines. Boxes can also connect to computers via VNC, SPICE, and Quemu.

Boxes uses qemu-kvm, libvirt-glib, and spice-gtk to allow users to easily manage virtual machines and connect to remote machines.

Features of Boxes

  • Create, access, and manage, local virtual machines.

  • Connect to remote machines (called connections) via SPICE, Quemu, or VNC protocols.

  • Select and create favorite virtual machines or connections.

How do I create a virtual machine in Boxes?

Download an ISO image of an operating system that will be used in the virtual machine.

Launch Boxes from the application launcher, super key, or terminal. Boxes icon

Click the New Button. New button location within the GUI in Gnome Shell

Read the introduction and click continue in the upper right hand corner.

Select the ISO Image file that was previously downloaded to the Downloads folder, otherwise click select a file to find an ISO image file located somewhere else.Boxes will try to auto-detect ISO files in your Downloads folder, but it may not always be successful. ISO Image Selection within Gnome Shell

Boxes will then auto create settings, they can be edited by clicking the customize button. Customize button location within Gnome Shell

Memory and Disk allocation can be changed via the customize menu. Customize Memory and Hard drive Space within Gnome Shell

After you are done allocating memory and hard drive space to the virtual machine click on the back button Customize back button within Gnome Shell

Click Create. Create button within Gnome Shell

The new virtual machine will now boot.

How do I connect to other computers in Boxes?

Note

Boxes Supports three protocols: SPICE, Qemu, and VNC.

Open Boxes.

ClickNew. New button location within the GUI in Gnome Shell

Click Enter URL. Enter URL location within the GUI in Gnome Shell

Enter the IP address or hostname of the remote machine you want to connect to. Remember to specify the protocol in the field. For example to connect to a computer via VNC at IP Address 192.168.1.115, you would type in vnc://192.168.1.115. Enter URL to the computer you want to connect to in Gnome Shell

Click Continue. Continue button location connect to a different computer in Gnome Shell

The Review screen will now show you the type (protocol) and host (either hostname or IP address) of the remote connection. Review Screen example in Gnome Shell

Click Create. Create button location in Gnome Shell

The Remote connection is now available in the selection screen. example selection screen in Gnome Shell after vnc connection is created

How do I change the settings of a machine in boxes?

  • On the listing of machines, right click on the machine and then choose Properties. Right Click menu in Boxes in Gnome Shell

  • If the machine is running, click on the menu button, the button with three lines, also known as the hamburger button Menu button location on a running VM on Gnome Shell

Once you have accessed the settings menu you can change various items of the virtual machine:

  • Sharing the clipboard

  • Resize Guest

  • Redirect New USB Devices to the Virtual Machine

  • Redirect currently plugged in USB devices to the Virtual machine

  • Create Snapshots of the Virtual Machine from the snapshot heading

  • Restart, If the Virtual machine is running, you can use the Restart button, to restart it. The Restart button is to the immediate left of the red Force Shutdown button on the system tab.

To exit the settings menu, press the X button X it is located in the upper right hand corner of the menu bar. This will close the settings screen and return you to your previous screen. Close button location for VM settings in Gnome Shell

How do I delete a box?

Deleting a box is simple and it can be done two ways:

Open Boxes.

Right Click the Box or Boxes you wish to delete. Close button location for VM settings in Gnome Shell

click the Delete button. Close button location for VM settings in Gnome Shell

Click the X on the confirmation notification if you are sure you have deleted the correct boxes. Or click UNDO if you made a mistake.

Open Boxes.

Click the checkmark button on the top right hand of the menu bar. Check mark button location in Gnome Shell

Select the boxes you wish to delete Multiple device selection checkmarks in Gnome Shell

After the Boxes have a check mark on them that you wish to delete, click the Delete button on the bottom bar. Multiple device deletion button in Gnome Shell

Click the X on the confirmation notification if you are sure you have deleted the correct boxes. Or click UNDO if you made a mistake.

Boxes Tips and Tricks

  • Boxes uses libvirtd and many libvirtd commands can be used for Boxes virtual machines.

  • Boxes keeps the virtual machine disks or images in:`/home/$USER/.local/share/gnome-boxes/images`

Advanced Commands in Boxes

You can start and stop virtual machines from the command line as well as using the boxes interface .

  • virsh shutdown or` virsh reboot` will use ACPI to shutdown or reboot the virtual machine.

  • virsh destroy will mimic if you pulled the power from a running virtual machine.

  • virsh start will power on or start the virtual machine.

Warning

Do not use the virsh destory often or corruption and data loss will occur!
